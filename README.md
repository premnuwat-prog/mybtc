# My BTC Cost

เว็บแอปส่วนตัวสำหรับดูต้นทุน BTC จากประวัติการซื้อขายบน Bitkub — ดึงข้อมูลจาก **หน้าเว็บ** โดยตรง ไม่ต้องใช้ API key

## วิธีใช้งาน

```bash
npm install
npm start
```

แล้วเปิด http://localhost:4780

## ปุ่ม "ดึงข้อมูลจาก Bitkub" ทำงานอย่างไร

1. ระบบเปิดหน้าต่าง Chrome (โปรไฟล์แยก เก็บไว้ในโฟลเดอร์ `browser-profile/`)
2. ไปที่หน้า https://www.bitkub.com/history — **ครั้งแรกให้ล็อกอิน Bitkub เองในหน้าต่างนั้น** (มีเวลา 6 นาที) ครั้งถัดไประบบจะจำ session ไว้ ไม่ต้องล็อกอินซ้ำ
3. สคริปต์จะดักจับข้อมูลประวัติการซื้อขายที่หน้าเว็บโหลด (และอ่านตารางบนหน้าจอเป็นแผนสำรอง) แล้วบันทึกลง `data/trades.json` โดยไม่บันทึกรายการซ้ำ — กด Sync กี่ครั้งก็ได้

## แผนสำรอง: นำเข้า CSV

ถ้าการดึงอัตโนมัติมีปัญหา (Bitkub เปลี่ยนหน้าเว็บ) ให้ใช้ปุ่ม **Export** ในหน้า history ของ Bitkub ดาวน์โหลดไฟล์ CSV แล้วกดปุ่ม "นำเข้า CSV" ในแอปแทน

## การคำนวณ

- ต้นทุนเฉลี่ยแบบถัวเฉลี่ย (average cost): ซื้อ = เพิ่มจำนวนและต้นทุน, ขาย = ตัดต้นทุนตามสัดส่วน และคิดกำไรที่ขายแล้ว (realized) แยกไว้
- ราคาล่าสุดมาจาก ticker สาธารณะของ Bitkub (ไม่ต้องล็อกอิน) อัปเดตทุก 1 นาที

## เวอร์ชันออนไลน์ (GitHub Pages)

เปิดจากเครื่องไหนก็ได้ที่ **https://premnuwat-prog.github.io/mybtc/** — ต้องใส่รหัสผ่านก่อนถึงจะเห็นข้อมูล

- ข้อมูลเทรดถูกเข้ารหัส AES-256-GCM ก่อนขึ้น GitHub (ไฟล์ `docs/data.enc`) ถอดรหัสในเบราว์เซอร์เท่านั้น — คนที่เปิด repo เห็นแค่ไฟล์เข้ารหัส
- รหัสผ่านอยู่ในไฟล์ `.encrypt-pass` บนเครื่องนี้ (ไม่ถูก commit) — เปลี่ยนได้โดยแก้ไฟล์นี้แล้วเผยแพร่ใหม่
- **วิธีอัปเดตข้อมูลขึ้นเว็บ**: กดปุ่ม "ดึงข้อมูลจาก Bitkub" แล้วตามด้วยปุ่ม "⬆ เผยแพร่ขึ้นเว็บ" ในแอปบนเครื่องนี้ (หรือรัน `npm run publish-web`)
- ราคาสดบนเว็บออนไลน์เรียกจาก Bitkub ตรงๆ ในเบราว์เซอร์ ส่วนข้อมูลกราฟ on-chain มี GitHub Action อัปเดตให้อัตโนมัติทุกวัน ~08:45 น.

## หมายเหตุ / แก้ปัญหา

- ข้อมูลทั้งหมดอยู่ในเครื่องคุณเท่านั้น (`data/` และ `browser-profile/`) — อย่าแชร์สองโฟลเดอร์นี้ให้ใคร เพราะมี session ล็อกอินอยู่
- ถ้าดึงข้อมูลแล้ว "ไม่พบรายการเทรด" ระบบจะเก็บข้อมูลดิบไว้ที่ `data/raw/` — เอาไฟล์นั้นมาให้ Claude ช่วยปรับตัวแปลงข้อมูลได้เลย
- ต้องมี Google Chrome (หรือ Microsoft Edge) ติดตั้งอยู่ในเครื่อง
